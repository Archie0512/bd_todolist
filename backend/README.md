# 工作任务清单 (Tasks.md)

基于开源项目 [Tasks.md](https://github.com/BaldissaraMatheus/Tasks.md) 搭建的任务管理看板。

## 快速开始

1. 双击 \启动.bat\ 启动服务
2. 浏览器会自动打开 http://localhost:8080
3. 用完后关闭命令行窗口或双击 \停止.bat\ 停止服务

## 看板结构

| 分类 | 说明 |
|------|------|
| 当月已完成-琐碎任务 | 当月已完成的零碎任务（如密码重置、打印机修复等） |
| 未完成任务 | 按系统区分的未完成任务（开发/运维） |
| 长期任务 | 需要较长时间完成的项目型任务 |

## 任务内容格式

任务内容支持 Markdown 语法，并支持以下特殊标记：

### 标签
在内容开头添加标签：
`
[tag:ERP] [tag:开发]

任务描述...
`

### 截止日期
`
[due:2026-07-31]

任务描述...
`

### 示例
`
[due:2026-07-25] [tag:ERP] [tag:开发]

## 问题描述
ERP系统报表导出时中文乱码

## 进度
- [x] 定位问题
- [ ] 修复代码
- [ ] 测试验证
`

## 数据存储

所有任务数据存储在 \	asks/\ 目录下，以 Markdown 文件形式保存。
配置文件（标签颜色等）存储在 \config/\ 目录下。

### 备份
直接复制 \	asks/\ 和 \config/\ 目录即可完成备份。

## 技术信息

- 前端: SolidJS + Vite
- 后端: Node.js + Koa
- 数据存储: Markdown 文件
- 运行环境: Node.js 18+
